# DiscordBlocks Electron v3.0
A Google Blockly (like MIT Scratch) to Discord.js compiler, built on Electron

## Why?
Electron is fun

## Open

0. `git clone`
0. `cd discord-blocks-electron`
0. `npm i`
0. `npm start`
